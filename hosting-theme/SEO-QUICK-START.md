# Hostorio Theme - SEO Quick Start (30 Minutes)

## Step-by-Step: Get Your Site Ranking on Google

### **Step 1: Install Rank Math SEO (5 minutes)**

1. Log in to **WordPress Admin Panel**
2. Go to **Plugins > Add New**
3. Search: **"Rank Math"**
4. Click **Install Now** > **Activate**
5. Follow the setup wizard:
   - Select "Business" for site type
   - Allow Google Search Console connection
   - Complete setup

---

### **Step 2: Submit Sitemap to Google (10 minutes)**

#### **Option A: Automatic (Recommended)**
Rank Math auto-submits to Google Search Console:
1. WP Admin > **Rank Math > Sitemaps**
2. Note your sitemap URL
3. Go to [Google Search Console](https://search.google.com/search-console)
4. Check "Sitemaps" section
5. Your sitemap should be there automatically!

#### **Option B: Manual Submission**
1. Get your sitemap URL from Rank Math
   - Usually: `https://yourdomain.com/sitemap_index.xml`

2. Go to [Google Search Console](https://search.google.com/search-console)

3. Select your website property:
   - If you don't have one, click "Add property"
   - Enter: `https://yourdomain.com`
   - Verify ownership (follow Google's steps)

4. Click **Sitemaps** (left sidebar)

5. Click **Add/test sitemap** button

6. Paste your sitemap URL:
   ```
   sitemap_index.xml
   ```

7. Click **Submit**

8. Wait for green checkmark (24-48 hours)

### **Step 4: Add Image Alt Text (5 minutes)**

This helps Google understand your images and improves accessibility.

1. Go to **Media Library**
2. Click on each image
3. Add **Alt Text** (example):
   - Hosting image: "VPS hosting server with cloud infrastructure"
   - Email image: "Professional business email interface"
   - Logo: "Hostorio web hosting company logo"

**Or do this per-page:**
1. Edit page/post
2. Click image
3. Add **Alt text** in image settings

---

### **Step 5: Install Speed Plugin (3 minutes)**

Page speed is a major Google ranking factor.

1. **Plugins > Add New**
2. Search: **"WP Super Cache"**
3. Install & Activate
4. Go to **WP Super Cache > Easy**
5. Click **"Caching On (Recommended)"**

Done! Your site will now load 2-3x faster.

---

### **Step 6: Check Your Work (2 minutes)**

#### **Test #1: Sitemap Submitted?**
- Go to [Google Search Console](https://search.google.com/search-console)
- Select your property
- Click **Sitemaps**
- You should see your sitemap with status

#### **Test #2: Page Speed**
- Go to [Google PageSpeed Insights](https://pagespeed.web.dev)
- Enter: `https://yourdomain.com`
- Should see **Mobile: 80+** and **Desktop: 90+**

#### **Test #3: SEO Optimization**
- WP Admin > **Rank Math Dashboard**
- Check "Content AI" score for each page
- Target: Green status (90+)

## 🚨 Common Issues & Fixes

### **Sitemap Not Submitting?**
```
Solution:
1. Go to WP Admin > Rank Math > Sitemaps
2. Copy the sitemap URL
3. Check URL in browser (should return XML)
4. Manually submit to Google Search Console
```

### **Pages Not Ranking?**
```
Solution:
1. Ensure focus keyword in title
2. Write 300+ word content
3. Add internal links
4. Wait 4-8 weeks (patience needed!)
5. Check Rank Math score is 80+
```

### **Low Page Speed Score?**
```
Solution:
1. Activate WP Super Cache
2. Install Imagify (image compression)
3. Remove unused plugins
4. Delete old posts/media
5. Use Rank Math > Modules > Performance
```

### **Google Search Console Errors?**
```
Solution:
1. Click on error type
2. Click "Affected pages"
3. Check each page
4. Fix 404s with Redirection plugin
5. Resubmit page in Search Console
```
