================================================================================
  BUSINESS EMAIL PAGE SETUP
================================================================================

Business Email is set up as a WordPress page template with 3 email packages.
Each package has a different product ID.

────────────────────────────────────────────────────────────────────────────────
HOW TO CREATE THE BUSINESS EMAIL PAGE
────────────────────────────────────────────────────────────────────────────────

1. Go to WordPress Admin > Pages > Add New

2. Enter Page Title:
   Business Email

3. Leave the content editor empty (or add additional info if desired)

4. In the right sidebar, find "Page Attributes" section

5. Select Template dropdown and choose:
   "Business Email"

6. Click Publish

CURRENCY SWITCHING
────────────────────────────────────────────────────────────────────────────────

Email prices support the currency switcher:
- Users can toggle BDT ↔ USD at the top of the page
- Prices automatically convert based on exchange rate in Theme Settings
- Exchange rate: Settings > Hosting Theme Settings > Currency > Exchange Rate

────────────────────────────────────────────────────────────────────────────────
CUSTOMIZING EMAIL PACKAGES
────────────────────────────────────────────────────────────────────────────────

To modify email packages:

1. Edit: hosting-theme/page-templates/template-business-email.php

2. Find the $email_packages array (around line 48)

3. Modify package details:
   - 'title'       → Package name
   - 'description' → Short description
   - 'price_bdt'   → Price in BDT
   - 'price_usd'   → Price in USD
   - 'features'    → Array of feature strings
   - 'order_url'   → Cart link with product ID
   - 'is_popular'  → true/false for "Most Popular" badge

4. Save and push changes
